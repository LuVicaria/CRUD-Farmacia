# CRUD-Farmacia

Este projeto foi criado como parte da atividade da Generation Brasil.  
A branch `configurando-o-projeto` contém a configuração inicial do projeto Spring Boot, incluindo:

- Estrutura do projeto
- Dependências no `pom.xml`
- Configuração do banco de dados no `application.properties`
- Teste básico de inicialização da aplicação

## Tecnologias utilizadas
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
